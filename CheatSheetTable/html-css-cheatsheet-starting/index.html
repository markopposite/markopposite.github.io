CSS Box Model Reference

Position Attributes

Attribute Values Description
position: Relative, etc... Sets how content is positioned in a document
display: Block, inline-block, etc... Sets element as inline or box
padding: Top, right, bottom, left Space within a box surrounding the content
border: 1px solid color Line element between the padding and the margin
margin: TRBL, in px or percent Space surrounding the outside border of a box

Examples

Attribute Description
border: 2px solid blue Prints a 2 px solid line around the pad of a content box
margin: 0 0 10px A 10 px margin on the bottom
margin: 10px 0 0 A 10 px margin on top, overrides bottom margin above
margin: 0 auto Sets the left-right margins automatically
padding: 0 50px Sets a left-right pad inside the content box
